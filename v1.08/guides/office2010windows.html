How to Install Microsoft Office 2010 on Windows

Microsoft Office 2010 is an older, now out of support, version of the Microsoft Office suite.

👍 Advantages

  • Less telemetry and bloat compared to modern versions.
  • More responsive and less resource intensive applications.
  • Fully compatible with all documents created on Office 2019 and later.
  • No longer automatically updates.

👎 Disadvantages

  • None.


📥 Downloading & Installation

🔗 Download from Get into PC

  1. Visit this link and click the "Download 64 Bit x64" button at the bottom of the page.
  2. Wait for the download to begin.
  3. Once complete, extract the archive using 7-zip and the password "123".

📀 Installation

  1. Open the extracted folder and run the "setup.exe" file.
  2. Click the "Customize" box and uncheck any applications you would not like installed.
  3. Wait for the installation to complete.
  4. Click "Close".


🔑 Activation

  1. Open Windows Security, click "Virus & threat protection", then disable "Real-time protection".
  2. Visit this link to download KMS_VL_ALL_AIO.
  3. Extract the archive using 7-zip and the password "2021".
  4. Right click the .cmd file in the newly extracted folder and click "Run as administrator".
  5. Click "Yes" to any UAC prompts.
  6. Type 2 to "Install Activation Auto-Renewal".
  7. Wait for the process to complete then close the application.
  8. Delete all downloaded files and restart Windows.
