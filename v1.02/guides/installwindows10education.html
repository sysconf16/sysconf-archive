How to Install Windows 10 Education 21H1

Windows 10 Education is a SKU which offers less built-in advertising and tracking in comparison to consumer SKUs, including Home and Pro.

  • Semi-annual channel feature updates.
  • Best hardware compatibility, supporting all features of 30-series Nvidia graphics cards and the new AMD scheduler.
  • Supports the Microsoft Store for easy installation of UWP applications.
  • Supports DRM (Play Ready and Widevine) with pre-installed Microsoft Edge.
  • Extended support until 2025.

  • Cannot be tweaked and optimised as well as Windows 10 Enterprise LTSC 1809.
  • Deeper baked Microsoft spyware which somewhat still phones home after basic privacy tweaks.
  • Greater system restrictions compared to SKUs such as Enterprise or Pro.

ğŸ“€ Creating Bootable Media

  1. On a spare Windows computer, visit this link to download the Windows 10 Media Creation Tool.
  2. Insert the 8GB+ USB into the computer and ensure that it is blank of any data.
  3. Open the Windows 10 Media Creation Tool and accept the license agreement.
  4. Check "Create installation media (USB flash drive, DVD or ISO file) for another PC" and click "Next".
  5. Click "Next" through the "Select language, architecture, and edition" screen.
  6. Check "USB flash drive" and click "Next".
  7. Ensure that the correct USB is selected from the removable device list and click "Next".
  8. Wait for the tool to download and prepare the USB stick. Exit once complete.
  9. Create a new file in the root of the USB stick named "ei.cfg".
  10. Open the file in Notepad and paste the following text:

    11. [EditionID]
    Professional
    [Channel]
    Retail

  11. Save the file and remove the USB stick from the spare computer.

ğŸ”Œ Booting from USB

  1. Ensure that the computer you wish to install to is turned off.
  2. Insert the USB stick into a USB port. Some motherboards can have trouble booting from USB 3.0 ports so it is advised to use a USB 2.0 port where possible.
  3. Push the power button while simultaneously pressing the boot menu key on the keyboard repeatedly. Please see a list of boot menu keys below.
  4. Select the USB stick from the boot menu list. A Windows logo should appear.

ğŸ–´ Installing to Disk

  1. Click "Next" on the languages and preferences screen.
  2. Click the "Install Now" button.
  3. If prompted to enter a product key, click "I don't have a product key".
  4. Select "Windows 10 Education" from the list and click "Next".
  5. Check the "I accept the license terms" box and click "Next".
  6. Click the "Custom: Install Windows only (advanced)" box.
  7. Select and delete all partitions on the drive you wish to install to so that only a single "Drive X Unallocated Space" remains.
  8. Click "Drive X Unallocated Space" and click "Next.
  9. Wait for the installation to complete. The system may restart and this is normal.


2ï¸âƒ£ Final Stages of Installation

ğŸ“¦ Out-of-Box-Experience (OOBE)

  1. From the "Let's start with region. Is this right?" screen, progress through the wizard until prompted to connect to a network.
  2. On the "Let's connect you to a network" screen, click "I don't have internet" in the bottom left and "Continue with limited setup".
  3. Choose a username and password for the system.
  4. Disable all when given the privacy settings options.
  5. Click "Not now" when prompted with "Let Cortana help you get things done".
  6. The desktop should now be visible.

ğŸ”‘ Activation

  1. Open Windows Security, click "Virus & threat protection", then disable "Real-time protection".
  2. Visit this link to download MAS.
  3. Extract the archive using 7-zip and the password "1234".
  4. Navigate to the following file ".\MAS_1.4_Password_1234\MAS_1.4\Separate-Files-Version\Activators\HWID-KMS38_Activation\HWID_Activation.cmd".
  5. Right click the .cmd file and click "Run as administrator".
  6. Click "Yes" to any UAC prompts.
  7. Wait for the process to complete then close the application.
