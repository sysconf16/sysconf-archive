How to Install Windows 10 Enterprise LTSC 1809

Windows 10 Enterprise LTSC 1809 is a special SKU which offers very little built-in advertising and tracking in comparison to other SKUs. It is designed for specialist systems where little maintenance is necessary.

👍 Advantages

  • Lightest, most privacy respecting edition of Windows 10 after tweaks and optimisations.
  • No Microsoft Store, UWP applications or DRM.
  • No semi-annual feature updates.
  • Extended support until 2029.

👎 Disadvantages

  • Lacks some modern hardware compatibility.


🔧 Tools Required

  • Spare Windows computer with an internet connection.
  • Blank 8GB+ USB stick.

⏲️ Installation Time

  • To the Desktop | 30mins - 1 hour.
  • Additional Tweaks and Configuration | 1 - 2+ hours.

1️⃣️ First Stages of Installation

🔌 Booting from USB

  1. Ensure that the computer you wish to install to is turned off.
  2. Insert the USB stick into a USB port. Some motherboards can have trouble booting from USB 3.0 ports so it is advised to use a USB 2.0 port where possible.
  3. Push the power button while simultaneously pressing the boot menu key on the keyboard repeatedly. Please see a list of boot menu keys below.
  4. Select the USB stick from the boot menu list. A Windows logo should appear.

🖴 Installing to Disk

  1. Select the correct language and keyboard settings on the languages and preferences screen, then click "Next".
  2. Click the "Install Now" button.
  3. If prompted to enter a product key, click "I don't have a product key".
  4. Select "Windows 10 Enterprise LTSC" from the list and click "Next".
  5. Check the "I accept the license terms" box and click "Next".
  6. Click the "Custom: Install Windows only (advanced)" box.
  7. Select and delete all partitions on the drive you wish to install to so that only a single "Drive X Unallocated Space" remains.
  8. Click "Drive X Unallocated Space" and click "Next.
  9. Wait for the installation to complete. The system may restart and this is normal.


2️⃣ Final Stages of Installation

📦 Out-of-Box-Experience (OOBE)

  1. From the "Let's start with region. Is this right?" screen, progress through the wizard until prompted to connect to a network.
  2. On the "Let's connect you to a network" screen, click "I don't have internet" in the bottom left and "Continue with limited setup".
  3. Choose a username and password for the system.
  4. Disable all when given the privacy settings options.
  5. Click "Not now" when prompted with "Let Cortana help you get things done".
  6. The desktop should now be visible.

🔑 Activation

  1. Open Windows Security, click "Virus & threat protection", then disable "Real-time protection".
  2. Visit this link to download KMS_VL_ALL_AIO.
  3. Extract the archive using 7-zip and the password "2021".
  4. Right click the .cmd file in the newly extracted folder and click "Run as administrator".
  5. Click "Yes" to any UAC prompts.
  6. Type 2 to "Install Activation Auto-Renewal".
  7. Wait for the process to complete then close the application.
